# Techway Solution Website

## Prerequisites
- Node.js v18+ (https://nodejs.org)
- pnpm: `npm install -g pnpm`

## Run locally
```
pnpm install
pnpm run dev
```
Open http://localhost:5173

## Build for production
```
pnpm run build
```
Output goes to the `dist/` folder — upload it to any static host.
